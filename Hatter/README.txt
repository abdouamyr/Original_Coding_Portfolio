HATTER 

Description: 
2d shooter employing (self-designed) sprite animations and basic physics. Use of keyboard controls, employment of pause function, creation of basic AI, collision section. Uses the “Phaser” game environment. (built in javascript to run in HTML). 

Controls/Gameplay: 
Use the directional keys for movement/jumping. Use the “f” key to shoot. Use the “e” key to pick up health and ammo. Use the “q” key to swap weapons. Use the “p” key to pause the game. The player wins once all the enemies have been killed, or loses if killed by one of the enemies.
