COLOR PLUS 

Description: 
Demonstration of mouse and number key input in tandem. More advanced use of 2d array. Use or Mathf.random function to add unpredictably to gameplay. Employment of functions to recognize player-created patterns. 

Controls/Gameplay: 
Use number keys to determine which row a colored cube is played into. Use mouse to move colored cubes into plus formations: either of the same color cubes, or plusses that include one of each of the five colors. If the player does not select a row for the cube, a random white cube is destroyed. Cubes can only move one space at a time.
